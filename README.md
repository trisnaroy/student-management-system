# 🚀 Student Management System

A simple console-based Student Management System built using Python.

This project allows users to manage student records using basic CRUD operations (Create, Read, Update, Delete).

---

## 📌 Features
- Add new student
- View all students
- Update student information
- Delete student
- Simple menu-based interface

---

## 🛠️ Technologies Used
- Python 3

---

## ▶️ How to Run

Open terminal in project folder and run:

```bash
cd student-management-system
python main.py
